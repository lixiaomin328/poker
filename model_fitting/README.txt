\f0\fs24 \cf0 This package contains function needed to fit behavioral data for poker game with cursed equilibrium model, QRE model and level k model.\
\
All .m files are functions needed. \
solveQRE.m and modelfitting.m are two main functions one wants to run.\
\

We used processed density data in nObsPoker.mat to fit the model. This data needs to be updated easily if new data are collected. It is the pooled strategy over different card profiles.


.mat files are sample data which can always be replaced by your own data.\
All data are transformed into 20 units bins density map for estimation. \
}
